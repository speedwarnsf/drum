# Drum — AI Drum Practice App

## What This Is
Drum practice app with 40 rudiments, spaced repetition, metronome. Built for Dustin who is learning drums.

## Stack
- Next.js + TypeScript + Tailwind
- Supabase
- ElevenLabs audio (metronome clicks, practice chimes)

## Key Features
- 7 pages, all working
- 40 rudiments with spaced repetition
- Metronome
- Practice tracking

## Design
- No rounded corners
- No emojis
- Clean, focused practice UI — minimal distractions
